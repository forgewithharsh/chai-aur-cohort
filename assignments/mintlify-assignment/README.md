# Mintlify Website Clone

A desktop-first recreation of the Mintlify marketing and documentation landing page.  
This project focuses on layout structure, visual hierarchy, and design fidelity using only HTML and CSS.

---

## Recreated Sections

- Header / Top Navigation Bar  
- Hero Section with CTA  
- Trusted By / Customer Logos  
- Feature Highlights & Product Sections  
- Enterprise Features  
- Customer Stories  
- Final Call-To-Action  
- Footer with multi-column links and status info
